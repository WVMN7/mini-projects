PULSEMIX MUSIC WEBSITE
======================

Bu loyiha siz so‘ragan zamonaviy music website.

Nimalar bor:
- Zamonaviy glassmorphism dizayn
- Animatsiyalar
- Search
- Filter chiplar
- Queue panel
- Pastda music player
- Visualizer
- Demo playlist (videodan ko‘ringan nomlar asosida)
- Papkani yuklash funksiyasi

Qanday ishlatish:
1) Papkani oching
2) index.html ni oching
3) Yoki VS Code + Live Server bilan ishga tushiring
4) Sayt ochilgach “Papkani yuklash” tugmasini bosing
5) Music fayllaringiz turgan papkani tanlang
6) Sayt audio fayllarni o‘zi library ga joylaydi

Muhim:
- Demo rejimdagi treklar faqat nomlar bilan turadi
- Haqiqiy playback uchun o‘z audio papkangizni yuklashingiz kerak
- Chrome brauzerda yaxshi ishlaydi

Agar xohlasangiz keyingi bosqichda men sizga:
- admin panel
- login/register
- artist page
- category page
- trending section
- lyrics block
- dark/light ultra premium versiya
ham tayyorlab beraman.
