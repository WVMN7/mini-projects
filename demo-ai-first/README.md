# AI Media Studio

Modern website for:
- image generation
- video generation
- Google search based inspiration / idea lookup

## Setup

```bash
npm install
copy .env.example .env
```

Then open `.env` and add your keys.

## Run

```bash
npm run dev
```

or

```bash
npm start
```

Open:

```text
http://localhost:3000
```

## Notes
- Image generation uses Gemini image model.
- Video generation uses Veo model.
- Google search panel uses Programmable Search / Custom Search JSON API.
